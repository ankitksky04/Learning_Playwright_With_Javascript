# Learn Playwright Batch 2x

A hands-on JavaScript learning repository designed for beginners and QA/automation learners. The content is organized in progressive chapters so you can build fundamentals step by step before moving to automation frameworks.

## Repository Overview

This repository focuses on JavaScript core concepts through small runnable files and practice-oriented examples.

Topics covered include:
- JavaScript basics and setup
- Identifier rules and best practices
- `var`, `let`, `const`, scope, and hoisting
- Literals and data values
- Operators and expression behavior
- Conditional logic with `if/else` and `switch`
- User input handling in Node.js
- Looping constructs and iteration practice
- Arrays and common array methods
- Functions, parameters, return values, closures, and higher-order functions
- String methods and conversion
- Objects, destructuring, spread, getters, setters, and references

## Chapter Guide

1. `chapter_01_Basics`
Introduction and starter JavaScript files.

2. `chapter_02_Javascript_Concepts`
Core JavaScript concept foundations.

3. `chapter_03_Identifier_Literals`
Identifier naming rules and valid/invalid examples.

4. `chapter_04_Javascript_Concepts`
Variables, scope, functions, and hoisting behavior.

5. `chapter_05_Literal`
Number, string, template literals, `null`, and `undefined`.

6. `chapter_06_Operator`
Arithmetic, comparison, logical, ternary, increment/decrement, and related exercises.

7. `chapter_07_If_else`
Conditional flow with real-world and interview-style problems.

8. `chapter_08_Switch_Statement`
`switch`, `break`, `default`, grouped cases, and practical examples.

9. `chapter_09_UserInput`
Taking user input in Node.js via `readline` and synchronous prompt flows.

10. `chapter_10_Loops`
Looping concepts and iteration-based examples.

11. `chapter_11_Arrays`
Array creation, access, add/remove operations, searching, iteration, transform methods, sorting, slicing, concatenation, and array checks.

12. `chapter_12_Funtions`
Function declarations, parameters, return values, function expressions, arrow functions, IIFE, default/rest/spread parameters, scope, closures, higher-order functions, and pure functions.

13. `chapter_13_Strings`
String creation, properties, indexing, search/check methods, substring/slice behavior, transformations, splitting/joining, and conversion.

14. `chapter_14_Objects`
Object literals, property access, primitive vs reference behavior, object examples, destructuring, spread, getters/setters, `this`, and `let` vs `const` with objects.

## Prerequisites

- Node.js (LTS recommended)
- VS Code (recommended editor)

Check your environment:

```bash
node -v
npm -v
```

## How To Run Any File

From the project root:

```bash
node chapter_01_Basics/01_Basics.js
node chapter_06_Operator/30_Operator.js
node chapter_09_UserInput/69_Node_readline.js
node chapter_11_Arrays/83_Arrays.js
node chapter_12_Funtions/96_Functions.js
node chapter_13_Strings/118_Strings.js
node chapter_14_Objects/124_Objects.js
```

## Recommended Learning Flow

- Follow files in numeric order (`01_`, `02_`, `03_`, ...).
- Complete one chapter before moving to the next.
- Run every file and observe the output.
- Modify values/operators/conditions and re-run to understand behavior deeply.

## Practice Tips

- Keep notes for tricky areas like hoisting and loose vs strict equality.
- Recreate examples from memory in a new file.
- Write one additional custom example for each concept.

## Purpose

This repository is practice-first. Many files are intentionally focused and short so you can iterate quickly, experiment confidently, and build strong JavaScript fundamentals for future Playwright automation work.
